Scripting Basics (Luau)

Scripts let you respond to events, read and set properties, and call methods on instances in the DataModel.

ConceptExample
Variables & Typeslocal count:number = 0 local name:string = "Player"
Functionslocal function add(a:number,b:number):number return a+b end
Events (RBXScriptSignal)workspace.Part.Touched:Connect(function(hit) print("Touched by", hit.Name) end)
Serviceslocal DSS = game:GetService('DataStoreService')
