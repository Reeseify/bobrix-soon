Luau Overview

Luau is Roblox’s scripting language derived from Lua 5.1 with improvements like gradual typing, type inference, and performance optimizations. It powers all scripts in Studio.

-- Type annotations export type PlayerProfile = { coins:number, name:string } local function loadProfile(id:number): PlayerProfile return {coins=0,name="Guest"} end
