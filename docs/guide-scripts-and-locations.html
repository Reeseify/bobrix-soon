Script Types & Locations

Server Script runs on the server and can access server-only services. Client LocalScript runs on a player’s device and can access input, camera, and GUI. ModuleScript returns functions/tables for reuse by other scripts.

-- Module example local M = {} function M.greet(n) return "Hello, "..n end return M
