Getting Started

This page shows how to add a simple script in Studio, print text, and connect an event. Scripts are plain text containers that run Luau, Roblox’s language.

-- In StarterPlayer > StarterPlayerScripts local Players = game:GetService('Players') print('Hello from Bobrix!') Players.PlayerAdded:Connect(function(plr) print('Joined:', plr.Name) end)

As you progress, move logic to appropriate script types (server vs client) and use ModuleScripts to share code between scripts.

Why Luau? It’s a fast, safe dialect of Lua designed for Roblox, with gradual typing and analysis tools.
